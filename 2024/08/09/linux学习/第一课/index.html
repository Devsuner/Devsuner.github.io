linux的目录结构

基本介绍

  1. “/“根目录
  2. root home bin etc lib lib64 sbin usr var tmp proc sys dev boot media mnt opt selinux srv tmp var
    1. /bin 存放二进制可执行文件,Binary的缩写,记录着常用的命令
    2. /sbin 存放系统管理员使用的管理程序:super user–>系统管理员用到的系统文件
    3. /home 存放普通用户的主目录–>存放普通用户的主目录
    4. /root 存放root用户的主目录–>系统管理员的目录
    5. /lib 存放系统最基本的动态链接共享库，其作用类似于Windows里的DLL文件。几乎所有的应用程序都需要用到这些共享库
    6. /etc 存放系统管理所需要的配置文件和子目录:比如安装mysql数据库
    7. /usr 存放系统应用程序，比较重要的目录/usr/local 本地系统管理员软件安装目录（安装系统级的应用）。这是另一个称为“Unix System Resources”的共享数据的目录，并不是“User”的缩写。它是从UNIX系统来的，在Linux系统中，它包含了绝大多数应用程序和文件。/usr/bin目录下存放着用户命令文件
    8. /boot 存放启动Linux时使用的一些核心文件，包括一些连接文件以及镜像文件
    9. /proc 存放系统内存的映射，访问这个目录来获取系统信息勿动
    10. /dev 存放设备文件
    11. /media 存放可移动介质设备，比如软盘和光盘
    12. /mnt 存放临时挂载的文件系统
    13. /tmp 存放临时文件
    14. /var 存放运行时需要改变数据的文件(即不断扩充的数据文件)，比如各种日志文件
    15. /mnt 系统管理员安装临时文件系统的安装点
    16. /opt 额外安装的可选应用程序包所放置的位置。默认情况下，这个目录是空的 安装软件所用目录,约定熟成
    17. /selinux 存放安全策略相关的文件
      18./user/local一般通过编译源码的方式安装的目录
